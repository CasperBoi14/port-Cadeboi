# Casper Adeboi - Future Portfolio

This is a futuristic portfolio website template designed for **Casper Adeboi**.

## Technologies Used
- **HTML5**: Semantic structure.
- **CSS3**: Futuristic styling, Glitch effects, Flexbox/Grid, CSS Variables.
- **JavaScript (Vanilla)**: Custom cursor, Scroll animations, Text scrambling interactions.

## How to Run
Simply open `index.html` in your web browser.

## Features
- **Custom Cursor**: A sci-fi cursor that follows your movement.
- **Glitch Text**: "Hello World" and titles have a glitch effect.
- **Scroll Animations**: Sections fade in as you scroll. Skill bars fill up.
- **Interactive details**: Hover over the "I AM CASPER" title to see a text decoding effect.

## Customization
- **Colors**: Change the colors in `styles.css` under the `:root` variables (e.g., `--primary-color`).
- **Content**: Edit `index.html` to update text, links, or add more projects.
